# ğŸ“Œ Prefix Sum

## â“ Why Do We Need Prefix Sum?

Prefix Sum is used when we need to handle multiple range sum queries efficiently.

If we calculate the sum for every query using a loop, the time complexity becomes high when the number of queries increases.

To optimize this, we preprocess the array once and store cumulative sums in a separate array.  
After preprocessing, each range sum query can be answered in constant time.

### â± Time Complexity

- Preprocessing: O(n)  
- Each Query: O(1)
